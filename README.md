# Mineração de Dados ✨

Repositório organizado para reunir atividades, exercícios e exemplos desenvolvidos ao longo das aulas.

## O que tem aqui? 🌷

- Exercícios resolvidos
- Exemplos de código para prática
- Scripts SQL e arquivos de apoio
- Atividades separadas por pasta

## Estrutura do repositório 🎀

```text
ex-06-03/
ex-10-04/
ex-13-03/
ex-17-04/
ex-20-02/
ex-27-02/
```

Cada pasta guarda um conjunto de atividades, com arquivos `.py` e, em alguns casos, arquivos auxiliares como `.txt`.

## Objetivo 💗

Registrar as atividades realizadas em aula para que o professor consiga acompanhar meu progresso ao longo das aulas.
